# Dotfiles

## Installation

### Step 1: Clone the repository

```sh
git clone https://github.com/mariuskilian/dotfiles.git ~/.dotfiles
```

### Step 2: `cd` into the folder

```sh
cd ~/.dotfiles
```

### Step 3: Apply desired configs

For a specific config:

```sh
stow <config>
```

For all configs:

```sh
stow */
```
